# LINE 創作工作區

同一個本機工作區支援：

- LINE 主題
- LINE 靜態貼圖
- LINE 表情貼

三種模式共用 `rembg/isnet-general-use` Alpha 去背（含 alpha matting 邊緣細化，模型無法載入時退回 `u2netp`）、YuNet 人臉位置偵測（含眼睛關鍵點，模型缺失時退回 Haar cascade）、無人臉時的主體偵測（Alpha 剪影優先，否則用頻譜殘差顯著性）、專案 JSON 與後端 ZIP 驗證。
人臉功能只用於裁切與排序，不辨識身分。眼睛關鍵點只在當次請求中用來對齊裁切位置。當次分析會產生不具名的視覺相似指紋，只用來避免連續挑到近似構圖，不會寫入磁碟或作身分辨識。

YuNet 模型檔位於 `creator_tool/models/face_detection_yunet_2023mar.onnx`；rembg 模型第一次使用時會自動下載到 `~/.u2net/`。

## 啟動

```bash
node server.js
```

- 主題：`http://127.0.0.1:3000/`
- 靜態貼圖：`http://127.0.0.1:3000/packs.html?mode=sticker`
- 表情貼：`http://127.0.0.1:3000/packs.html?mode=emoji`
- 專業去背工具：`http://127.0.0.1:3000/creator_tool/`

## 靜態貼圖

- 支援 8、16、24、32、40 張。
- 可直接拖放、選擇多張圖片或選擇整個資料夾；匯入素材寫入 `.workspace/imports/`。
- 草稿模式使用 `u2netp`，高品質模式使用 `isnet-general-use` 與 alpha matting。
- 去背以原圖 SHA-256、模型與參數快取至 `.cache/cutouts/`，重開專案可直接沿用。
- 批次處理會顯示進度、預估時間並可取消。
- 智慧指派會先把可用素材各用一次，再考慮重複；沒有可靠人臉的貓狗或物件也會納入候選。
- 主體類型可選「自動／人物／寵物或物件」；自動模式只採用高信心人臉，寵物則依完整 Alpha 輪廓構圖。
- 合照會列出偵測到的人物，預設全部納入聯合安全框；逐一選擇後，系統會以人臉種子在去背前景內分割人物，移除未選人物並按格位保存結果，同時從原始 Alpha 取回所選人物相連的頭髮／帽子輪廓。
- 同一素材若已有草稿與高品質去背，會比較兩份遮罩；差異過大的格位以橘／紅框及警示符號提示人工檢查。
- 紅色高風險草稿會自動改用該張的高品質去背快取；若快取不存在，批次流程只升級該張，不拖慢其餘素材。
- 可在預覽上拖曳平移、滾輪縮放，每格構圖會隨專案儲存。
- 臉部特寫會在完整原圖去背後，以主臉定位並保護相連的頭髮／帽子／頭套輪廓；孤立遮罩雜點不會撐大構圖，AI 預設縮放為 94%。
- 靜態貼圖文字不再固定占用 24% 高度；自動排版會比較底部中央、左下與右下的主體重疊量，也可逐格手動指定位置。
- 「一鍵建立初稿」會串接智慧指派與快速去背；正式輸出時自動升級高品質。
- 編輯狀態會自動儲存至瀏覽器，重開分頁後自動復原。
- 內容圖輸出 370×320 PNG，四周預留約 10px。
- 自動從第 1 張生成 `main.png` 240×240 與 `tab_on.png` 96×74。
- 輸出：`line_stickers.zip`。

## 表情貼

- 支援 8～40 張一般表情貼。
- 內容圖輸出 180×180 PNG，檔名為 `001.png`～`040.png`。
- 自動從第 1 張生成 `tab_on.png` 96×74。
- 輸出：`line_emoji.zip`。

## 專業去背工具

- 支援拖放與多檔匯入。
- 橡皮擦與遮罩修整保留最近 20 步 Undo／Redo。
- 快捷鍵：`Cmd/Ctrl+Z` 復原，`Cmd/Ctrl+Shift+Z` 重做。

工作區會檢查格位、透明背景、可見內容、PNG 格式、尺寸、單檔 1 MB 與 ZIP 容量限制。
