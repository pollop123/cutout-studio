# LINE 真人照片主題素材說明

> 此文件保留主題模式的特定說明；靜態貼圖與表情貼請參閱根目錄 `README.md`。

## 視覺分配

- 商店與主題縮圖：`100039_0.jpg`（粉紅頭套照）
- 單人預設頭像：`100046_0.jpg` 生成的單人剪影
- 群組預設頭像：`100029_0.jpg` 生成的三人剪影
- 選單按鍵：九個選單各使用一張不同的去背臉部，無圓框，並在畫布內置中
- 密碼圖示：來自 `100027_0.jpg`、`100028_0.jpg`、`100032_0.jpg`、`100033_0.jpg` 的去背臉部
- iOS 聊天背景：`100041_0.jpg`（花海，完整照片置中）
- Android 聊天背景：`100041_0.jpg`（花海，完整照片置中）

OFF 圖示為半透明黑白，ON 圖示保留原色。密碼圖示不使用外框。

## 產生方式

```bash
python3 -m venv venv
venv/bin/pip install -r requirements.txt
venv/bin/python server.py
```

主題編輯器位於 `http://127.0.0.1:3000/`，去背與遮罩整理工具位於
`http://127.0.0.1:3000/creator_tool/`。兩個畫面共用同一個 Python 圖像處理後端。

主題編輯器的剪影以 Alpha 遮罩為唯一前景來源；如果原圖沒有透明背景，
請先按「AI 去背」再啟用剪影。

## 編輯器工作流程

- `儲存專案`：下載 JSON，包含所有指派、裁切參數與已修整的透明素材。
- `載入專案`：在原始圖片仍存在時回復工作進度。
- `還原原圖`：移除當前圖片的 AI 去背與手動擦除結果。
- `人臉輔助裁切`：只在本機偵測人臉位置與數量，不辨識身分、不建立人臉特徵資料庫。
- `智慧批次指派`：依圖片比例、解析度、人臉數量與人臉佔比選擇單人、群組與背景素材。

產生 ZIP 前，瀏覽器會先檢查 62 張檔案的檔名、尺寸、PNG 格式與選單背景透明度；
後端會再次檢查實際 ZIP，並在必要時壓縮聊天背景至 1 MB 以內。

輸出：

- `line_theme/`：展開的 PNG 素材
- `line_theme_interactive.zip`：主題編輯器產生的 ZIP

如需使用寫死素材配置的舊自動化流程，可執行 `node auto_generate_zip.js`，
其輸出為 `line_theme_scripted.zip`，不會覆寫互動編輯器的成品。

`generate_theme.py` 會進行比例縮放、人物焦點裁切、檔名與尺寸檢查，並確保聊天背景各自低於 1 MB。

## 規格備註

LINE 官方繁體中文摘要頁目前寫 60 張、選單 34 張，但詳細規格表實際列出了 18 張舊版選單圖與 18 張 iOS 26 選單圖。本專案依詳細表完整輸出 62 張，避免缺少新或舊版檔案。
